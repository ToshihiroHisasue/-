SQLの練習問題 第二部

ここでは発展的なSQLの練習問題を紹介します。

Sales 部門の従業員の ID と名前を取得する

SELECT ___, ___ FROM employees WHERE department = 'Sales';
解答: id, name

25歳から35歳の従業員のすべての情報を取得する

SELECT ___ FROM employees WHERE ___ BETWEEN 25 AND 35;
解答: *, age

HR 部門の従業員の給与を 500 増やす

UPDATE employees SET salary = ___ WHERE department = 'HR';
解答: salary + 500

従業員の部署情報を NULL に設定する (ID が 3 のみ)

UPDATE employees SET ___ = ___ WHERE id = 3;
解答: department, NULL

部署ごとに従業員数をカウントする

SELECT department, ___(*) FROM employees GROUP BY ___;
解答: COUNT, department

平均給与が 6000 を超える部署を取得する

SELECT department, AVG(salary) FROM employees GROUP BY department ___ AVG(salary) > 6000;
解答: HAVING

従業員と部署情報を結合して取得する

SELECT e.name, d.department_name FROM employees e ___ departments d ON e.department_id = d.id;
解答: INNER JOIN

給与が 7000 を超える従業員の一時的なテーブルを作成する

CREATE ___ TABLE high_salary_employees AS SELECT * FROM employees WHERE salary > 7000;
解答: TEMPORARY

hire_date カラムを従業員テーブルに追加する

ALTER TABLE employees ADD COLUMN hire_date ___;
解答: DATE

25歳未満の従業員を削除する

DELETE FROM employees WHERE ___;
解答: age < 25
