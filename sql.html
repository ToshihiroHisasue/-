SQLの練習問題

ここではSQLの練習問題を紹介します。

SQLテーブルからすべての列を取得する

SELECT ___ FROM SQL;
解答: *

SQLテーブルから名前と年齢を取得する

___ name, age ___ SQL;
解答: SELECT, FROM
　　

SQLテーブルから年齢が30以上の社員を取得する

SELECT * FROM SQL ___ age >= 30;
解答: WHERE
　　

SQLテーブルから給与の降順でデータを並べ替える

SELECT * FROM SQL ___ ___ salary ___;
解答: ORDER BY , DESC
　　

SQLテーブルの給与の合計を取得する

SELECT ___(salary) FROM SQL;
解答: SUM(salary)
　

studentsテーブルを作成し、カラムとしてid（整数）、name（文字列）、age（整数）を追加する

___ TABLE students (id ___, name ___, age ___);
解答: CREATE, INT, VARCHAR(50), INT);

studentsテーブルに新しい行（id: 1, name: 'Alice', age: 20）を挿入する

___ ___ students (id, name, age) ___ (1, 'Alice', 20);
解答: INSERT INTO, VALUES;

studentsテーブルのname列をfull_nameに変更する

___ TABLE students ___ COLUMN name ___ full_name;
解答: ALTER, RENAME, TO;

studentsテーブルからageが20未満の行を削除する

___ ___ students WHERE age < 20;
解答: DELETE FROM;

studentsテーブルを削除する

___ TABLE students;
解答: DROP;
